# service-worker-app
CRA(Create React App)에 서비스 워커를 적용했습니다.

개발자가 코드를 수정하여 배포하면 사용자에게 알림을 주어 항상 최신 버전을 사용할 수 있도록 돕습니다.

## 1. 기능
* 메뉴를 클릭하여 페이지 이동 시 새로운 버전 존재 여부를 확인합니다.
* 새로운 버전이 존재하면 사용자에게 업데이트 알람을 보냅니다.
* 확인을 누르면 새로운 버전의 페이지를 보여줍니다.
